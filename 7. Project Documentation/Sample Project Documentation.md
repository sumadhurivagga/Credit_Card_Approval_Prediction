## Problem Statement
Manual credit card approval review is slow, inconsistent, and prone to bias. This project automates the decision using Machine Learning.

## Skills Applied
Xgboost, Machine Learning Algorithms, Artificial Intelligence, Decision Tree Learning, NumPy, Python, Scikit-Learn, Matplotlib

## Solution Summary
An ML classification model (XGBoost/Decision Tree) trained on applicant data predicts credit card approval status, deployed as a live Streamlit web app for instant, real-time predictions.

## Outcome
A fully functional, publicly deployed web application that takes applicant details and returns an instant Approved/Rejected prediction.
