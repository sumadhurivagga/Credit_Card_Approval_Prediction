# Project Demo Planning

## Demo Flow
1. Introduce the problem: manual, inconsistent credit card approvals
2. Walk through the solution architecture (input → preprocessing → model → prediction)
3. Live demonstration on the deployed Streamlit app with sample applicant data
4. Show model performance metrics (accuracy, precision, recall, F1-score)
5. Q&A / discuss scalability and future improvements

- Live deployed app: https://creditcardapprovalprediction-hrzqcjzayrde7zffub5zxc.streamlit.app/
