# Demonstration of Proposed Features

## Feature Walkthrough
1. **Input Form** — User enters applicant details (income, employment, family size, housing, credit history)
2. **Submit** — Data sent to the backend preprocessing + trained model
3. **Instant Prediction** — Approval/Rejection result displayed within seconds
4. **Accessible Anywhere** — Fully deployed on Streamlit Cloud, usable from any browser
