# Scalability & Future Plan

## Current Scope
A working ML-based Credit Card Approval Prediction model deployed as a live Streamlit web app.

## Scalability
- Model can be retrained on larger, more diverse datasets for better generalization
- Can be integrated into bank backend systems via a REST API instead of a standalone app
- Can handle concurrent requests when hosted on scalable cloud infrastructure

## Future Enhancements
- Add explainability (e.g. SHAP values) to justify why an application was approved/rejected
- Incorporate additional data sources (transaction history, credit bureau scores)
- Build an admin dashboard for banks to monitor approval trends
- Add authentication and secure data handling for production use
