# Technology Stack

| Layer | Technology |
|---|---|
| Programming Language | Python |
| Data Handling | Pandas, NumPy |
| Data Visualization | Matplotlib |
| Machine Learning | Scikit-Learn, XGBoost, Decision Tree Learning |
| Model Serialization | Pickle (.pkl) |
| Deployment / Web App | Streamlit, Streamlit Cloud |
| Version Control | Git & GitHub |

## Why this stack?
- **XGBoost & Decision Trees** — strong performance on structured/tabular data like credit applications, and easily interpretable for approval reasoning.
- **Scikit-Learn** — reliable preprocessing pipeline and baseline model comparisons.
- **Streamlit** — enables quick, browser-based deployment without needing a separate front-end/back-end setup, ideal for showcasing the model as a live demo.
